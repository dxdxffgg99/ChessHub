# ♟️ 체스 프로젝트

LiChess에서 영감을 받아 제작된 오픈소스 체스 플랫폼입니다.  
누구나 자유롭게 학습하고 기여할 수 있어요.

---

## 🚀 기능

- 실시간 체스 게임 (PvP / AI)
- 클라우드 또는 클라이언트 기반 Stockfish 분석
- 오픈소스 체스 로직 구현
- 반응형 UI (모바일 지원)

---

## 📦 기술 스택

- 백엔드: Cloudflare Workers + Durable Objects
- 엔진: WebAssembly 기반 Stockfish
- 저장소: Redis, D1 또는 PostgreSQL
